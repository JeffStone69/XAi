Description
-----------
A sleek, production-ready, single-file Streamlit dashboard that scans 40+
hand-selected stocks at the heart of the physical economy: Mining, Shipping,
Energy, Renewables, and related Technology (ASX + US markets).

It computes a proprietary risk-adjusted **Signal Score** that blends short-term
momentum, volume conviction, and volatility control. The result is a clean
leaderboard that highlights the strongest geo-supply alpha in real time.

The "Grok Thesis" tab lets you instantly call xAI's Grok models to transform
cold quant rankings into high-signal 2–5 day narrative intelligence.

Optimized, fixed, and enhanced by Grok (xAI) itself.

Key Features
------------
• Dynamic leaderboards (5d / 10d / 1mo horizons)
• Sophisticated risk-adjusted Signal Score (momentum × vol spike × volatility dampener)
• Beautiful dark-mode Plotly charts (price + volume) for the top names

     - Full ranked watchlist CSV
     - Interactive Brokers (IBKR) Top-10 watchlist import file

Installation & Quick Start
--------------------------
1. Download or clone: https://github.com/JeffStone69/XAi

2. Install dependencies:
   pip install streamlit pandas yfinance plotly requests numpy

3. Run the app:
   streamlit run geosupply_analyzer.py

4. (Highly recommended) Add your x.ai Grok API key in the sidebar to unlock
   the AI Thesis generator.
